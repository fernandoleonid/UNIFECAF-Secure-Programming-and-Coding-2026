# Secure Programming and Coding - UniFECAF

Repositório de apoio prático (hands-on) da disciplina **Secure Programming and Coding**.

A proposta é trabalhar segurança de software de forma aplicada, com exemplos vulneráveis e versões corrigidas, cobrindo fundamentos, vulnerabilidades comuns, ferramentas de automação e segurança em APIs/arquiteturas modernas.

## Trilha da Disciplina

O curso está organizado em **4 unidades** com foco prático (4 aulas por unidade, total de 16 aulas):

1. **Fundamentos de Desenvolvimento Seguro**  
Princípios de codificação segura, tratamento de erros, SDL/SSDLC/DevSecOps e Shift Left Security.
2. **Vulnerabilidades e Mitigações (OWASP, CWE, MITRE)**  
OWASP Top 10, injeções, proteção de dados sensíveis e autenticação/autorização segura.
3. **Ferramentas, Testes e Automação de Segurança**  
Análises SAST/DAST, integração em pipelines e geração de métricas/artefatos de segurança.
4. **Segurança em APIs e Arquiteturas Modernas**  
Boas práticas para APIs, tokens/sessões, hardening de endpoints e cenários de microserviços.

## Como Usar Este Repositório

1. Navegue por unidade e aula conforme o cronograma da disciplina.
2. Execute primeiro os exemplos vulneráveis para observar riscos reais.
3. Compare com as versões seguras/corrigidas para entender mitigação.
4. Replique os testes localmente e adapte os exemplos para novos cenários.
