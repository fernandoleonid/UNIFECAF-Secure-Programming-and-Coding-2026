## 💡 Principais Conceitos

### SDL (Security Development Lifecycle)
**Metodologia estruturada em 7 fases:**
1. Treinamento
2. Requisitos
3. Design
4. Implementação
5. Verificação
6. Lançamento
7. Suporte

### SSDLC (Secure SDLC)
**Conceito: Segurança integrada desde o início**

Princípios:
- Segurança desde o início
- Análise de riscos contínua
- Responsabilidade compartilhada
- Boas práticas estabelecidas (OWASP, CWE)
- Verificação e validação constantes
- Métricas de segurança

### DevSecOps
**Cultura + Automação: Segurança no Pipeline CI/CD**

Foco:
- Desenvolvimento ágil
- Pipeline automatizado (Plan → Code → Build → Test → Deploy → Operate)
- Testes de segurança contínuos (SAST, DAST, SCA)
- Detecção precoce de vulnerabilidades
- Ferramentas: SonarQube, OWASP ZAP, Snyk, Trivy, HashiCorp Vault
